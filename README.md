# StarShift-App

## Overview
A design concept of a real-time constellation and astronomy learning app designed to help users 
understand what they see when they look up.

## Process
<ul>
  <li>User research</li>
  <li>Low-fidelity Prototype</li>
  <li>Wireframing</li>
  <li>High-fidelity UI Design</li>
</ul>
